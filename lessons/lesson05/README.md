# Lesson 5: TickTask定时任务

## 课程目标
学习如何使用TickTask创建定时任务，了解定时器在任务调度中的应用。

## 学习内容
- 如何定义一个TickTask
- 如何设置定时器间隔
- 定时任务的执行逻辑
- 定时任务的自动停止

## 任务说明
本课程演示了一个定时计数器的场景。你需要按照TODO注释的提示，逐步取消注释来完成代码。

## 运行方法
```bash
cd lesson05
go run main.go
```

## 知识点
1. **TickTask结构体**：基于定时器的任务，支持周期性执行
2. **GetTickInterval方法**：设置定时器执行间隔
3. **Tick方法**：定时器触发时执行的逻辑
4. **自动停止**：任务可以自己决定何时停止
5. **资源管理**：定时器资源的正确创建和清理
